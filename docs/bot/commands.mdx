## أوامر Slash Commands

### 🎫 أوامر التذاكر
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `/ticket-panel` | إنشاء لوحة التذاكر | Administrator |
| `/setup` | إعداد نظام التذاكر | Administrator |

### 📋 أوامر الإدارة
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `/help` | عرض قائمة الأوامر | جميع المستخدمين |

## أوامر Prefix ($)

### 🛠️ أوامر الإعداد
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `$setup` | عرض إعدادات النظام | Administrator |
| `$setup category <ID>` | تعيين قسم التذاكر | Administrator |
| `$setup sendchannel <ID>` | تعيين قناة إرسال التذاكر | Administrator |
| `$setup role <@role>` | إضافة رتبة دعم | Administrator |
| `$setup removerole <@role>` | إزالة رتبة دعم | Administrator |
| `$setup adminlog <ID>` | تعيين قناة سجلات الإدارة | Administrator |
| `$setup log <ID>` | تعيين قناة سجلات التذاكر | Administrator |
| `$setup rating <ID>` | تعيين قناة التقييمات | Administrator |
| `$setup publish` | نشر زر إنشاء التذاكر | Administrator |

### 🎫 أوامر التذاكر
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `$ticket <الموضوع>` | إنشاء تذكرة جديدة | جميع المستخدمين |

### 📚 أوامر المساعدة
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `$help` | عرض قائمة الأوامر | جميع المستخدمين |

### 🔧 أوامر الصيانة
| الأمر | الوصف | الصلاحية |
|---|---|---|
| `$cleardb` | تنظيف قاعدة البيانات | Administrator |

## أمثلة على الاستخدام

### إعداد النظام
```
$setup category 123456789012345678
$setup sendchannel 987654321098765432
$setup role @Support
$setup publish
```

### إنشاء تذكرة
```
$ticket مشكلة في الحساب
```

### الحصول على المساعدة
```
$help
```

## ملاحظات مهمة

:::warning تحذير
- جميع أوامر الإعداد تتطلب صلاحيات Administrator
- تأكد من صحة معرفات القنوات والأقسام
- استخدم `$setup` بدون معاملات لعرض الإعدادات الحالية
:::

:::tip نصيحة
- يمكن استخدام أوامر Prefix للتحكم السريع
- أوامر Slash Commands توفر واجهة أكثر تفاعلية
- استخدم `$help` للاطلاع على جميع الأوامر المتاحة
:::
