## للطاقم الجديد

### 1. التحقق من الصلاحيات
تأكد من أن لديك إحدى الصلاحيات التالية:
- **إداري**: صلاحيات كاملة على النظام
- **رقابي**: إدارة التذاكر والمراجعة
- **مسؤول**: تشغيل يومي وتقارير
- **مبرمج**: وصول API وإعدادات متقدمة

### 2. الوصول للوحة الويب
1. انتقل إلى [ksa1980.lol](https://ksa1980.lol)
2. اضغط على "Sign In" 
3. سجل الدخول عبر Discord OAuth
4. سيتم توجيهك تلقائياً للوحة التحكم

### 3. التعرف على النظام
- اطلع على **الأدلة الداخلية** حسب دورك
- تعرف على **أوامر البوت** المتاحة
- استكشف **لوحة الويب** ومزاياها

## للطاقم الموجود

### تحديث سريع
- تم تحديث النظام مع مزايا جديدة
- تحسينات في الأداء والأمان
- واجهة محدثة للوحة الويب
